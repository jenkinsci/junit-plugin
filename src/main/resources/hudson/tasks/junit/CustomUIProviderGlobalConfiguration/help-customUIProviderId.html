Select a custom UI provider to use globally for all JUnit test results.

When set, this provider will be used to render test results for all jobs that don't have a specific custom UI provider configured. Custom UI providers allow external plugins to provide alternative visualizations, dashboards, or custom representations of test results.

Select "Default UI" to use the standard Jenkins JUnit test results interface.

Note: Individual jobs can override this global setting by specifying their own custom UI provider in the job configuration.
